# Bat Bakery

A spooky-themed bakery website HTML structure for night owls, insomniacs, and dessert rebels who crave fresh bakes after dark.

## Project Overview

This project contains HTML-only versions of all Bat Bakery website pages.

### HTML Structure Overview

- **index.html** - Homepage with hero section and featured products
- **about.html** - About the bakery and founding story  
- **products.html** - Full menu and product catalog
- **services.html** - Baking classes and custom orders
- **contact.html** - Contact information and location

## HTML Features Implemented

- `<header>` - Navigation and branding
- `<nav>` - Main navigation menu
- `<main>` - Primary content area
- `<section>` - Content sections (hero, featured products)
- `<article>` - Individual product cards
- `<footer>` - Contact info and links

## Content Structure

## HTML Template Structure

Basic HTML5 template used across all pages:

html
<!DOCTYPE html>
<html lang="en">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>Page Title | Bat Bakery</title>
</head>
<body>
    <header>
        <!-- Simple text logo and navigation -->
    </header>
    
    <main>
        <!-- Page-specific content -->
    </main>
    
    <footer>
        <!-- Contact info and links -->
    </footer>
</body>
</html>
